# FileManager
 

This is a simple application created in **Visual Studio 2022**.

## Requirements
To run this project, you need:

- .NET 8 SDK
- Visual Studio 2022 **or** Visual Studio Code
- Git

Verify .NET installation:
 bash
dotnet --version

## Requirements
### Visual Studio 2022
1. Clone the repository:
2. Open Visual Studio 2022:
 Select Open a project or solution
 Open the project/solution from the cloned repository
3. Run the application using:
 Run button, or F5

### Visual studio Code
1. Open the project folder in Visual Studio Code
2. Open the integrated terminal:
3. Clone project:
   git clone https://github.com/jendruleka/FileManager.git
2. Make sure you are in the project root directory  
   cd directory root
4. Run app
   dotnet run
